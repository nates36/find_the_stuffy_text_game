# find_the_stuffy_text_game

A very simple and barebones text-based guessing or hidden object game/function. You're given a list of places where the stuffy (stuffed animal) is hidden in a room. You get 3 attempts or guesses to find the stuffy. If you find it in 3 guesses, you win. If you don't find it, you lose.

Modules used:
random
